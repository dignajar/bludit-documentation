Title: Installationsanleitung
Position: 3
Content:

1. Die neueste Version von Bludit von https://s3.amazonaws.com/bludit-s3/bludit-builds/bludit_latest.zip herunterladen.
2. Das ZIP-Archiv entpacken.
3. Den Inhalt des ZIP-Archivs auf den Server laden.
4. Im Browser die Domain und eventuell das Verzeichnis aufrufen, in dem sich die Dateien befinden.
5. Auf der Seite Bludit Installer die gewünschte Sprache der Installation wählen und "Next" klicken.
6. Passwort für den Benutzer "admin" und eine E-Mail-Adresse eingeben, Installation abschliessen.

<div markdown="1" class="note">
<div class="note-title">WICHTIG</div>
Es empfiehlt sich, nach der Installation die entsprechende Zeitzone einzustellen im Administrationsbereich unter *Einstellungen -> Sprache und Zeitzone -> Zeitzone*.
</div>

<div markdown="1" class="note">
<div class="note-title">Hinweis</div>
Um die Dateien auf den Server zu laden, kannst du einen FTP-Client verwenden. Vgl. dazu [FTP](../verschiedenes/ftp).
</div>
